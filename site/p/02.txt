Cara Mengatur Kecepatan TrackPoint Thinkpad Serta
Disable TouchPad Pada Arch Linux

<span id="read-time">0</span> min | <span id="word-count">0</span> character 
____________________________________________________

TL;DR: Pengaturan sensivitas TrackPoint dan disable 
       Touchpad. Beberapa id mungkin berbeda-beda :

       $ xinput list | grep -i "trackpoint" 
         | awk -F'id=' '{print $2}' 
         | awk '{print $1}'
       11 
       $ xinput set-prop 11 "libinput Accel Speed" 
         -0.66
       $ xinput list | grep -i "touchpad" 
         | awk -F'id=' '{print $2}' 
         | awk '{print $1}'
       10 
       $ xinput disable 10

Jika teman-teman menggunakan laptop bawaan Lenovo
(dulu IBM) pasti sudah tidak asing lagi dengan
"pentil merah" yang berada tepat di tengah-tengah
keyboard.
Ya, itu yang dinamakan trackpoint. Lalu, apa saja
fungsi dari trackpoint?
  
1. Navigasi 

2. Klik

3. Scroll

Yang mungkin tidak hanya itu. Saya lebih sering
menggunakan trackpoint di banding mouse atau
touchpad. Terutama untuk menggeser-geser sebuah
window. Penggunaan trackpoint ini sebenarnya cukup
meningkatkan efisiensi penggunaan keyboard tanpa
harus mengangkat tangan untuk menggunakan mouse atau
sekedar elus-elus touchpad untuk melakukan gerakan
navigasi.

Jika di gambarkan adalah hanya jari telunjuk sebelah
kanan saya yang aktif untuk menggerakan kursor dan
sisanya tetap aktif berada di atas keyboard tanpa
terdistraksi oleh apapun [1].

Lalu bagaimana cara mengatur kecepatan trackpoint
pada Arch Linux?

Teman-teman harus membuat sebuah file pada 
/etc/udev/rules.d/10-trackpoint.rules [2] dengan 
perintah : 

  $ sudo touch /etc/udev/rules.d/10-trackpoint.rules

kemudian isi file tersebut dengan baris ini :

  ACTION=="add", KERNEL=="serio2", 
  SUBSYSTEM=="serio", DRIVERS=="psmouse", 
  ATTR{sensitivity}="300", ATTR{speed}="300"

Sebelum menuju ke tahap selanjutnya, perhatikan pada
kedua nilai ini :
  
  ATTR{sensitivity}="300"
  ATTR{speed}="300"

Nilai tersebut dapat di ubah sesuai dengan 
preferensi masing-masing, saya membuat nilai 
tersebut menjadi :

  ATTR{sensitivity}="100"
  ATTR{speed}="100"

Tujuannya agar tidak terlalu cepat dan lebih akurat.

Atau jika ingin lebih mudah, gunakan xinput dengan :

  $ xinput list
    ⎡ Virtual core pointer                  id=2
      [master pointer  (3)]
    ⎜   ↳ SynPS/2 Synaptics TouchPad        id=10
          [slave  pointer  (2)]
    ⎜   ↳ TPPS/2 IBM TrackPoint             id=11
          [slave  pointer  (2)]
    ⎣ Virtual core keyboard                 id=3
      [master keyboard (2)]
        ↳ Virtual core XTEST keyboard       id=5
          [slave  keyboard (3)]

Dari keluaran tersebut, perhatikan id pada 
TPPS/2 IBM TrackPoint yang bernilai 11 :

  $ xinput list-props 11
    Device 'TPPS/2 IBM TrackPoint':
	  Device Enabled (183):	1
    libinput Accel Speed (332):	-0.660000 

Perhatikan pada keluaran akhir.

  xinput set-prop 11 "libinput Accel Speed" -0.66 

Nilai dari -0.66 dapat teman-teman sesuaikan dengan 
preferensi masing-masing. 

Trackpoint sudah, lalu bagaimana dengan disable 
touchpad? Apa teman-teman masih ingat dengan 
keluaran yang di hasilkan dari xinput tadi?

  $ xinput list
    ⎡ Virtual core pointer                  id=2
      [master pointer  (3)]
    ⎜   ↳ SynPS/2 Synaptics TouchPad        id=10
          [slave  pointer  (2)]

TouchPad memiliki id 10, jadi cukup gunakan 
perintah ini :

  $ xinput disable 10

karena saya menggunakan i3wm [3] maka saat ini 
saya menambahkan skrip ke dalam 
~/.config/i3/config agar di muat setiap kali 
i3wm di jalankan.
