Kembali berpindah static site generator [1] ke 
static site lainnya―― 

Source code asli berasal dari $/dylanaraps (saat ini
sudah tidak aktif. Cerminan dapat di akses pada 
situs resmi Kiss Linux [2]) artikel-artikel di tulis 
menggunakan Vim [3] ASCII [4] pada postingan ini di
buat dengan cinta serta hosting menggunakan layanan 
GitHub [5] dan domain dari Rumahweb [6].

Blog ini berisi artikel yang sedikit-demi-sedikit 
atau selesai di teliti secara epistemik (tentu akan
kontradiktif) Tentang Unix, secuil filsafat dan 
banyak lagi. 

Beberapa bagian nya lagi mencoba mengambil sudut 
pandang baru, menghubungkan ide-ide secara intuitif 
tanpa menghilangkan sumber asli.

Kalimat asing dan juga sumber daya eksternal bukan 
menggunakan gaya href [7] melainkan menggunakan 
footnotes [8] alih-alih lebih bersih pada elemen 
&lt;pre&gt; [9].
