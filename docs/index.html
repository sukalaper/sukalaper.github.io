Bagaimana Cara Membaca Blog Ini?
________________________________________________________________________________

Ini tidak seperti static site lainnya sumber kode website saya ambil tanpa rasa 
tahu malu dari Dylan Araps.

blog
├── docs
│   └── # Berisi file .txt yang telah di convert ke .html
├── make # Berisi skrip yang digunakan untuk mengkonversi .txt ke .html dan 
│          akan membuat sebuah folder docs/
├── site
│   └── # Berisi file .txt yang nantinya akan di convert ke .html 
│         dengan ./make
└── template.html

Tidak ada Markdown, blok kode, pemberitahuan, tombol, kolom, panel. Tidak ada!

Menurut saya, ini bagus! Meningkatkan keterbacaan dan beberapa fungsi tanpa 
distraksi ketika menulis, membuat catatan atau melampirkan tutorial yang sahih.


Kutipan
________________________________________________________________________________

Keindahan subjektif merayakan permasalahan yang tak terbantahkan namun dihargai, 
menghadirkan konsekuensi logis yang bijak. Ide radikal melahirkan urgensi untuk 
mengambil langkah-langkah yang dibutuhkan, membuka pintu potensi pengalaman yang 
lebih baik. Tidak melulu tertutup dan berlagak konservatif, melainkan terbuka 
pada kemungkinan, karena jika menggunakan sesuatu itu salah maka selamanya saya 
tidak ingin benar.
